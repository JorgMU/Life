# Conway's Game of Life

A browser-based implementation of [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life) — a classic cellular automaton devised by mathematician John Horton Conway in 1970.

## About This Project

This project was created as an experiment with **GitHub Copilot** and simple **JavaScript programming**. The goal was to explore what Copilot can help build interactively, starting from a blank file and growing toward a functional, self-contained web page.

## Intentions

- Build a fully working Game of Life simulation that runs in the browser.
- Keep each page **self-contained** — all HTML, CSS, and JavaScript lives in a single `.html` file with no external dependencies.
- Use a resizable canvas so the grid adapts to different window sizes.
- Provide simple controls (start, stop, step, clear, randomize) accessible directly on the page.
- Iterate on features and polish through incremental milestones in collaboration with GitHub Copilot.

## Files

| File | Description |
|---|---|
| `TheGameOfLife.html` | The main self-contained Game of Life page |
| `Conversations/` | Session summaries and raw transcripts from Copilot sessions |

## How to Run

Open `TheGameOfLife.html` directly in any modern web browser. No build step or server required.
